Detailed Description

template<typename Distr, int N, ENABLE_IF_BASE_OF(Distr, dist::DistributionTag)>
class edda::VectorDistrArray< Distr, N, >

Array of vector distributions.

Note the distributions among vector components are independent, which is in the contrast of JointDistrArray
